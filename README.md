# Nuxt 3 Threads App (Prisma, Supabase, Tailwind CSS)

## Setup

Make sure to install the dependencies:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install
```

## Copy .env.example into .env file and fill it
